# Security policy

## Supported version

The latest deployed beta is supported. Facebook is an unstable external surface, so reproduce issues against the latest Actor build before reporting them.

## Reporting

Report security-sensitive issues privately to `spbotdel@gmail.com`. Do not publish access tokens, signed dataset URLs, personal data, or reproducible abuse instructions in a public issue.

## Scope

The Actor accesses only content Facebook exposes to logged-out public requests. It must not be modified or used to bypass privacy controls, authentication, checkpoints, rate limits, or other security measures.
